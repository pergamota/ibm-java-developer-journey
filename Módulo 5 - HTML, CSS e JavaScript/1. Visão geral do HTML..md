# HTML e HTML5

## O que é HTML

**HTML (HyperText Markup Language)** é uma linguagem de marcação criada originalmente para compartilhar documentos científicos.  
Ela utiliza **elementos** para estruturar o conteúdo de uma página web, rotulando partes como parágrafos, listas, tabelas, imagens, links, entre outros.

Os **elementos HTML** são os blocos de construção de uma página.

---

# Objetivos do HTML5

## Principais objetivos

- Definir uma **única linguagem padrão** chamada HTML5
    
- Garantir interoperabilidade com versões anteriores (HTML4 e XHTML1)
    
- Melhorar a marcação de documentos
    
- Incluir **novas marcações e APIs**, como:
    
    - Armazenamento na web
        
    - Áudio e vídeo
        
    - Gráficos e animações
        

---

# Recursos do HTML5

## O que o HTML5 oferece

# XML

## O que é XML

**XML (Extensible Markup Language)** é uma linguagem de marcação projetada para **armazenar e transportar dados**.

### Características

- Permite criar **linguagens personalizadas**
    
- Muito usada para troca de dados
    
- Mais rígida que HTML
    
- Voltada para dados, não para layout
    

---

# Estrutura Básica de um Documento HTML

## Principais tags

```
<!DOCTYPE html>
<html>
    <head>
        <title>Sample Page Title</title>
    </head>
    <body>
        <h1>Hello World</h1>
        <p>This is a paragraph.</p>
    </body>
</html>
```

### Explicação

- `<!DOCTYPE html>` → Define o tipo do documento
    
- `<html>` → Elemento raiz
    
- `<head>` → Metadados (título, scripts, estilos)
    
- `<body>` → Conteúdo visível ao usuário
    

---

# DOM (Document Object Model)

## O que é DOM

O **DOM** é uma representação em memória de um documento HTML.

### Características

- Organizado em forma de **árvore**
    
- Contém nós como:
    
    - DOCTYPE
        
    - `<head>`
        
    - `<body>`
        
    - Comentários
        
- Permite que scripts acessem e modifiquem elementos da página
    

---

# XHTML

## O que é XHTML

**XHTML (Extensible HyperText Markup Language)** é semelhante ao HTML, porém com regras **mais rigorosas**.

### Diferenças entre HTML e XHTML

- XHTML exige:
    
    - Tags em minúsculo
        
    - Todas as tags fechadas
        
    - Atributos com valores obrigatórios
        
    - Uso de aspas simples ou duplas
        
- HTML é mais flexível e tolerante a erros

# Scripts em HTML

## Uso de scripts

Scripts geralmente são escritos em **JavaScript** e podem ser:

- Embutidos:
    

`<script>     alert("Hello!"); </script>`

- Externos:
    

`<script src="script.js"></script>`

### Funções dos scripts

- Validação de formulários
    
- Manipulação dinâmica do DOM
    
- Alteração de imagens e estilos
    

---

## Segurança – Desativando Scripts

### Problema

Objetos incorporados sem o atributo `sandbox` podem executar scripts com permissões perigosas.

### Solução

Utilizar o atributo `sandbox`:

`<iframe src="example.html" sandbox></iframe>`

# Formulários HTML

## Estrutura básica

```
<form>
    <label for="name">Name:</label>
    <input type="text" id="name">

    <textarea rows="4" cols="30"></textarea>

    <button type="submit">Submit</button>
</form>
```
### Elementos do formulário

- `<form>` → contêiner do formulário
    
- `<input>` → entrada de dados
    
- `<textarea>` → texto longo
    
- `<button>` → ações
    
- `<label>` → acessibilidade
    
- `<fieldset>` → agrupamento lógico
