#  JavaScript – Visão Geral

## O que é JavaScript?

JavaScript permite:

- Criar websites interativos e dinâmicos
    
- Atualizar conteúdo sem recarregar a página inteira
    
- Criar animações e efeitos visuais
    
- Melhorar a experiência do usuário
    
- Tornar a navegação mais suave e responsiva
    

Quando você entra em um site:

1. O navegador baixa HTML e CSS
    
2. Também baixa o JavaScript
    
3. O JavaScript contém instruções que tornam a página interativa
    

---

# Manipulação do DOM

Permite:

- Alterar conteúdo
    
- Modificar estrutura
    
- Mudar estilo da página
    
- Inserir elementos
    
- Alterar atributos
    

Exemplo:

`document.getElementById("titulo").innerHTML = "Novo título";`

---

# Tratamento de Eventos

JavaScript responde a interações como:

- Clique
    
- Movimento do mouse
    
- Teclas pressionadas
    

Exemplo:

```
document.getElementById("botao").onclick = function() {
    alert("Botão clicado!");
};
```

---

# Como Incluir JavaScript

## Inline JavaScript

Dentro do HTML:

`<script>    console.log("Olá"); </script>`

Pode ser colocado no `<body>` ou no `<head>`, mas no `<head>` pode gerar resultados inesperados.

---

## External JavaScript (Melhor prática)

1. Criar arquivo `script.js`
    
2. Vincular ao HTML antes do `</body>`
    

`<script src="script.js"></script>`

Vantagens:

- Melhor organização
    
- Reutilização
    
- Código mais limpo

# Variáveis em JavaScript

JavaScript é dinamicamente tipada — não precisa declarar tipo.

## Regras de nome:

- Começar com letra, _ ou $
    
- Pode conter números
    
- Não pode começar com número
    

---

## var

Escopo de função.

`var nome = "Pedro";`

---

## let

Escopo de bloco (entre chaves).

`let idade = 18;`

Pode alterar valor.

---

## const

Valor constante.

`const PI = 3.14;`

Não pode ser reatribuída.

# Métodos de Arrays

## forEach()

Executa ação para cada elemento.

`frutas.forEach(function(fruta) {     console.log(fruta); });`

---

## map()

Cria novo array transformado.

`let numeros = [1,2,3]; let dobrados = numeros.map(n => n * 2);`

---

## filter()

Filtra elementos.

`let maiores = numeros.filter(n => n > 1);`

---

## reduce()

Reduz a um valor único.

`let soma = numeros.reduce((total, n) => total + n, 0);`

---

## find()

Retorna primeiro elemento que atende condição.

`let encontrado = numeros.find(n => n > 1);`

# DOM (Document Object Model)

## O que é?

Interface entre HTML e JavaScript.

Permite:

- Modificar conteúdo
    
- Alterar estrutura
    
- Atualizar estilos dinamicamente
    

---

## Estrutura Hierárquica

```
window
 ├── document
 ├── history
 ├── location
 ├── navigator
 ├── screen
 
```

### window

Objeto principal.

### document

Acesso aos elementos HTML.

### history

Histórico de páginas.

### location

Informações da URL.

### navigator

Informações do navegador.

### screen

Informações da tela.
