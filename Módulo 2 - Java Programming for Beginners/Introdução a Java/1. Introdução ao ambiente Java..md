
##  O que é Java

Java é uma linguagem de programação de **alto nível**, projetada para criar aplicativos que podem ser executados em **diferentes plataformas** sem a necessidade de reescrever o código.  
Isso é possível graças à **Java Virtual Machine (JVM)**, que executa o bytecode Java em qualquer sistema compatível.

Java é amplamente utilizada em:

- Aplicações corporativas
    
- Sistemas web
    
- Desenvolvimento mobile (Android)
    
- Computação em nuvem
    
- Microsservices


## Breve História do Java

- **1991** — Criado pela Sun Microsystems, inicialmente chamado **Oak**, voltado para dispositivos embarcados (TVs, videocassetes).
    
- **1998** — Lançamento do **Java 2**, trazendo melhorias como interfaces gráficas e a **Java Collections Framework**.
    
- **1999** — Introdução do **J2EE**, facilitando o desenvolvimento de aplicações corporativas de grande escala.
    
- **2004 (Java 5)** — Introdução de:
    
    - **Generics** (segurança de tipos)
        
    - **Annotations**
        
    - **Enhanced for-loop**
        
- **2006** — Java se torna **open source**, ampliando o ecossistema de bibliotecas.
    
- **2014 (Java 8)** — Introdução das **expressões lambda**, permitindo código mais limpo e funcional.
    
- **2018 (Java 11)** — Inclusão da **API HTTP** moderna e melhorias no **Garbage Collector**.


## Principais Vantagens do Java

- **Independência de plataforma** (Write Once, Run Anywhere)
    
- **Compatibilidade com versões anteriores**
    
- **Gerenciamento automático de memória (Garbage Collector)**
    
- **Forte foco em segurança**
    
- **Ampla adoção em projetos de longo prazo** (ex: LinkedIn)


## Ecossistema Java

### JDK, JRE e JVM

- **JDK (Java Development Kit)**  
    Conjunto completo para desenvolvimento Java (compilador, bibliotecas e ferramentas).
    
- **JRE (Java Runtime Environment)**  
    Ambiente necessário apenas para **executar** aplicações Java.
    
- **JVM (Java Virtual Machine)**  
    Máquina abstrata responsável por executar o bytecode Java.


### IDEs populares

- **Eclipse** — Open source, extensível por plugins
    
- **IntelliJ IDEA** — Recursos inteligentes e alta produtividade
    
- **NetBeans** — Open source, ideal para iniciantes
  
  
## Frameworks Java 
### Spring Framework

- Arquitetura **fracamente acoplada**
    
- **AOP (Aspect-Oriented Programming)**
    
- **Spring Boot** — configuração simplificada e servidores embutidos
    
- **Spring MVC** — arquitetura estruturada para aplicações web
  
## Hibernate

- Framework **ORM (Object-Relational Mapping)**
    
- Automatiza o mapeamento entre objetos Java e tabelas de banco de dados
  
  
## Build Tools

Ferramentas que automatizam compilação, dependências e empacotamento.

- **Maven** — ciclo de vida padronizado e gerenciamento de dependências
    
- **Gradle** — compilações incrementais, scripts em Groovy ou Kotlin  
  
  
## Servidores de Aplicação

- **Apache Tomcat** — container de servlets, ideal para aplicações web
    
- **JBoss** — suporta toda a pilha Java EE, usado em sistemas de grande porte
  
  
## Testes

- **JUnit** — testes unitários com anotações
    
- **TestNG** — testes baseados em dados e execução paralela
  
  
## Java na Nuvem

- **Spring Cloud** — ferramentas para aplicações nativas em nuvem
    
- Muito usado em **microsserviços** e arquiteturas distribuídas

## Programação Orientada a Objetos (POO)

Java é uma linguagem baseada em **objetos**, que representam entidades do mundo real.

## Conceitos principais

- **Classe** — modelo que define atributos e comportamentos
    
- **Encapsulamento** — protege dados internos do objeto
    
- **Herança** — reutilização de características entre classes
    
- **Polimorfismo** — mesma ação com comportamentos diferentes


## Características da Linguagem

- **Fortemente tipada**
    
- **Gerenciamento automático de memória**
    
- **Multithreading**
    
- **Biblioteca padrão extensa**
    
- **Alta segurança**
    
- **Estável e amplamente usada em IA e Machine Learning**  
  
  
## Observação Final

Java continua sendo uma das linguagens mais importantes do mercado devido à sua **estabilidade**, **robustez** e **ecossistema maduro**, sendo amplamente utilizada em aplicações corporativas, cloud e microsserviços.
