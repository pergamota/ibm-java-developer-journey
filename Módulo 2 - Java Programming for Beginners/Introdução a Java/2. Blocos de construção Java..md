
# Organização e Estrutura em Java

## Pacotes

### O que são pacotes

Pacotes são usados para agrupar classes e interfaces relacionadas. Eles ajudam a evitar conflitos de nomes iguais e facilitam o gerenciamento e a manutenção de projetos grandes.

### Criação de pacotes

Em Java, os pacotes são declarados usando a palavra-chave `package`.  
Exemplo:

`package com.example.myapp;`

Esse nome cria uma estrutura de diretórios correspondente, onde cada parte do nome representa uma pasta. Assim, a organização do código acompanha a hierarquia dos pacotes.

## Arquivos de origem e convenções

### Convenção de nomes de arquivos

Cada classe pública deve estar em seu próprio arquivo de código-fonte. O nome do arquivo deve ser exatamente igual ao nome da classe, seguido da extensão `.java`.

Por exemplo, se a classe se chama `Book`, o arquivo deve se chamar `Book.java`.

## Classes

### Conceito de classe

Uma classe é um modelo usado para criar objetos. Ela define atributos (dados) e métodos (comportamentos) que os objetos terão.

Por exemplo, em um aplicativo de compras, pode existir uma classe `Produto` com métodos como `calcularPreco` ou `aplicarDesconto`, que definem o comportamento dos objetos desse tipo.

## Métodos

### Função dos métodos

Métodos encapsulam funcionalidades dentro de uma classe. Eles representam ações que um objeto pode executar.

### Convenção de nomenclatura

Os métodos devem ter nomes descritivos e seguir um padrão consistente. Em Java, utiliza-se o padrão **camelCase**, no qual a primeira palavra começa com letra minúscula e as seguintes com letra maiúscula, como em `calcularPrecoFinal`.

## Ponto de entrada do programa

### Método principal

O ponto de entrada de uma aplicação Java é o método `main`, que normalmente está dentro de uma classe pública. Esse método é o primeiro a ser executado quando o programa inicia.

### Exemplo prático

Em um sistema de biblioteca, pode existir uma classe principal no pacote `com.example.library` com um método `main` responsável por testar o funcionamento do sistema.

Nesse contexto, cria-se um objeto do tipo biblioteca para armazenar livros, de forma semelhante ao uso de uma estrutura de dados. Um livro pode ser adicionado assim:

`library.addBook(new Book("1984", "George Orwell"));`

Para exibir todos os livros cadastrados, pode-se chamar:

`library.displayBooks();`

## Importação de classes

### Uso da palavra-chave import

Quando é necessário utilizar classes de outros pacotes, elas devem ser importadas no início do arquivo de código.

### Exemplo de importação

Primeiro, declara-se o pacote:

`package com.example.library;`

Depois, importam-se classes da biblioteca padrão do Java, como:

`import java.util.List;`  
ou  
`import java.util.ArrayList;`

Essas classes permitem a criação de listas dinâmicas para armazenar e gerenciar dados.

## Boas práticas de organização

### Organização do código

É importante utilizar nomes significativos e descritivos para classes, métodos e variáveis. Um código bem nomeado facilita a leitura, a manutenção e o trabalho em equipe.

### Estrutura de pacotes

Classes relacionadas devem ser agrupadas em pacotes coerentes, mantendo uma estrutura de pastas lógica e intuitiva.

## Tipos de dados em Java

### Tipos de dados primitivos

Os tipos de dados primitivos armazenam valores simples, possuem tamanho fixo e valores padrão definidos pela linguagem. Eles não podem receber o valor `null`.

Principais tipos primitivos:

`byte`: economiza memória e armazena valores entre -128 e 127  
`short`: usado para números inteiros pequenos, entre -32.768 e 32.767  
`int`: tipo inteiro mais utilizado  
`long`: usado para números inteiros muito grandes  
`float`: armazena números decimais com menor precisão  
`double`: armazena números decimais com maior precisão  
`char`: armazena um único caractere  
`boolean`: armazena valores verdadeiro ou falso

### Tipos de dados de referência

Tipos de referência são usados para apontar para objetos e estruturas de dados mais complexas.

`String`: representa uma sequência de caracteres  
`Array`: estrutura que armazena múltiplos valores do mesmo tipo  
`Classes`: servem como modelo para criar objetos  
`Objetos`: são instâncias de uma classe e contêm atributos e métodos
`Instância`: é quando a classe sai do papel e vira um objeto. Exemplo
`Carro meuCarro = new Carro();
* Carro = classe.
* meuCarro = objeto.
* new Carro() = instância.  

Objetos facilitam a reutilização de código, a organização do sistema e a manutenção do software.

### Interfaces

Interfaces funcionam como contratos que definem quais métodos uma classe deve implementar. Ao criar uma interface, apenas os métodos são declarados, sem implementação.

Por padrão, os métodos de uma interface são abstratos. O uso de interfaces torna o código mais flexível, limpo e fácil de manter, permitindo que diferentes classes compartilhem comportamentos comuns.

### Enumerações (Enum)

Enum é um tipo especial de dado que permite definir um conjunto fixo de valores nomeados. É muito utilizado para representar conceitos como dias da semana, estados ou cores.

## Operadores em Java

### Operadores básicos

Operadores são símbolos especiais que realizam operações sobre variáveis e valores.

Operadores aritméticos realizam operações matemáticas básicas.  
Operadores relacionais comparam valores, como `>`, `<`, `>=`, `!=`.  
Operadores lógicos combinam expressões booleanas, como `&&`, `||` e `!`.

### Operadores avançados

Operadores avançados são usados para otimização, manipulação de bits e operações de baixo nível.

Operadores bit a bit trabalham diretamente com dados binários.  
Operadores de deslocamento movem bits para a esquerda ou direita.  
Operadores de atribuição combinam operação e atribuição, como `+=` e `/=`.  
Operadores unários operam sobre um único valor, como o sinal `-`.  
O operador ternário é uma forma compacta de escrever uma estrutura `if/else`.

## Arrays e matrizes

### Arrays

Arrays são estruturas que armazenam vários valores do mesmo tipo.

Declaração de um array:

`int[] numeros;`

Inicialização de um array:

`numeros = new int[5];`  
ou  
`int[] numeros = {1, 2, 3, 4, 5};`

### Matrizes

Matrizes são arrays multidimensionais.

Exemplo de declaração de matriz bidimensional:

`int[][] matriz = new int[3][3];`
