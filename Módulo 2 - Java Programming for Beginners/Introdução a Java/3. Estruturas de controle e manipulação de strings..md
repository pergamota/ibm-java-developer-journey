
## Estruturas de decisão

### Switch

A estrutura `switch` verifica o valor de uma única variável em relação a múltiplos casos possíveis. Ela é útil para substituir cadeias longas de `if/else`, tornando o código mais limpo e legível.

O `default` é usado como uma opção alternativa, sendo executado quando nenhum dos casos (`case`) corresponde ao valor da variável analisada.

## Estruturas de repetição (Loops)

### For

O laço `for` é utilizado quando se sabe previamente o número de iterações que serão executadas. Ele é definido em três partes:

- Inicialização: define a variável contadora
    
- Condição: verifica se o loop deve continuar
    
- Incremento ou decremento: atualiza o contador após cada iteração
    

### While

O laço `while` é usado quando não se sabe o número exato de iterações. O código será executado enquanto a condição definida for verdadeira.

### Do-while

O `do-while` garante que o bloco de código seja executado pelo menos uma vez, pois a condição só é verificada após a primeira execução.

### Loops aninhados

Loops aninhados são laços de repetição dentro de outros laços. Eles são muito utilizados para trabalhar com estruturas de dados multidimensionais, como matrizes, por exemplo, um `for` dentro de outro `for`.

### Continue

A instrução `continue` é usada para pular a iteração atual do loop e seguir diretamente para a próxima iteração, sem interromper o laço completamente.

## Strings em Java

### Criação de Strings

Strings podem ser criadas de duas formas:

- Atribuindo um texto entre aspas duplas
    
- Utilizando a palavra-chave `new`
    

Exemplo:

`String texto = "Hello, World!";`

ou

`String texto = new String("Hello, World!");`

### Comprimento da String

O método `length()` retorna o número de caracteres de uma string, incluindo espaços.

Exemplo:

`int tamanho = texto.length();`

### Método charAt

O método `charAt()` é usado para acessar um caractere específico de uma string. O índice começa em 0.

Exemplo:

`char letra = texto.charAt(0);`

### Concatenação de Strings

A concatenação combina duas ou mais strings. Isso pode ser feito usando o operador `+` ou o método `concat()`.

Exemplo com operador `+`:

`String fullName = firstName + " " + lastName;`

Exemplo com `concat()`:

`firstName.concat(" ").concat(lastName);`

### Comparação de Strings

A comparação de strings deve ser feita usando o método `equals()`, que verifica se o conteúdo das strings é idêntico, diferenciando letras maiúsculas de minúsculas.

Exemplo:

`boolean igual = str1.equals(str2);`

### Extração de Substring

O método `substring()` permite extrair uma parte de uma string, informando o índice inicial e o final.

Exemplo:

`String sub = str.substring(5, 16);`

Nesse caso, o método retorna os caracteres a partir do índice 5 até o índice 15.

### Split

O método `split()` divide uma string em partes menores com base em um delimitador definido.

Exemplo:

`String csv = "apple,banana,cherry";`  
`String[] fruits = csv.split(",");`

Cada elemento do array resultante conterá uma parte da string original.

### Unir Strings (Join)

O método `String.join()` é utilizado para unir vários elementos de um array em uma única string, usando um separador.

Exemplo:

`String[] cores = {"Amarelo", "Verde", "Azul"};`  
`String resultado = String.join(", ", cores);`

### Strings imutáveis

Em Java, strings são imutáveis. Isso significa que, após serem criadas, não podem ser modificadas. Qualquer operação que pareça alterar uma string, na verdade, cria um novo objeto na memória.

### Métodos comuns de manipulação de Strings

`toUpperCase()`: converte todos os caracteres para maiúsculo

`toLowerCase()`: converte todos os caracteres para minúsculo

`trim()`: remove espaços em branco no início e no final da string

`replace()`: substitui caracteres ou partes da string por outros valores

## Pacotes e importações

### Importações

A importação permite que classes ou pacotes sejam utilizados no arquivo atual, tornando o código mais limpo e evitando a necessidade de escrever o nome completo do pacote sempre que uma classe for usada.

Exemplo de importação de uma classe específica:

`import shapes.Circle;`

Exemplo de importação de todas as classes de um pacote:

`import shapes.*;`

### Pacotes padrão do Java

`java.lang`: importado automaticamente, fornece classes básicas como `String`, `Math` e `Object`

`java.io`: permite operações de entrada e saída

`java.net`: oferece suporte à comunicação em rede

`java.sql`: facilita a conexão e execução de comandos em bancos de dados

`java.time`: fornece uma API moderna para manipulação de datas e horas

## Funções e métodos

### Funções

Uma função é um bloco de código que executa uma tarefa específica. Ela ajuda a organizar, reutilizar e tornar o código mais legível.

Em Java, uma função é definida especificando o tipo de retorno, o nome e os parâmetros.

### Métodos

Um método é uma função associada a uma classe. Ele define os comportamentos ou ações que os objetos dessa classe podem realizar.

A estrutura de um método inclui:

- Tipo de retorno
    
- Nome do método
    
- Parâmetros
    
- Corpo do método
    

### Diferença entre função e método

Funções são blocos de código independentes, enquanto métodos estão sempre associados a uma classe. Em Java, na prática, todo código executável está dentro de métodos.

### Parâmetros

Parâmetros são variáveis definidas entre parênteses que permitem passar dados para funções ou métodos durante a chamada.

### Sobrecarga de métodos (Method Overloading)

A sobrecarga de métodos permite definir vários métodos com o mesmo nome dentro de uma classe, desde que possuam parâmetros diferentes.

O Java escolhe automaticamente qual método executar com base nos argumentos fornecidos na chamada do método.
