
# Tratamento de Arquivos em Java

##  Conceito

O **tratamento de arquivos** permite **ler e gravar dados em arquivos** no computador, garantindo **persistência dos dados** mesmo após o programa ser encerrado.

Em Java, isso é feito principalmente com:

- `java.io`
    
- `java.nio`


# Pacote `java.io`

## Classe File (`java.io.File`)

A classe `File` **representa um caminho** para um arquivo ou diretório no sistema.

### Principais funcionalidades

- Criar arquivos
    
- Excluir arquivos
    
- Verificar propriedades
    
- Criar e excluir diretórios
    

### Exemplo – Classe File
```
import java.io.File;

public class FileExample {
    public static void main(String[] args) {
        File file = new File("example.txt");

        System.out.println(file.getName());
    }
}
```

---

## Escrita em Arquivos

### FileWriter

- Usado para **gravação de texto**
    
- Cria o arquivo se não existir
    
- Sobrescreve o conteúdo por padrão
    

### Exemplo – FileWriter

```
import java.io.FileWriter;
import java.io.IOException;

public class FileWriterExample {
    public static void main(String[] args) throws IOException {
        FileWriter writer = new FileWriter("output.txt");

        writer.write("Hello, world!");
        writer.close();
    }
}
```


---

### BufferedWriter

- Melhora o desempenho
    
- Ideal para **grandes volumes de dados**
    

### Exemplo – BufferedWriter

```
import java.io.BufferedWriter;
import java.io.FileWriter;
import java.io.IOException;

public class BufferedWriterExample {
    public static void main(String[] args) throws IOException {
        BufferedWriter writer = new BufferedWriter(new FileWriter("output.txt"));

        writer.write("Linha 1");
        writer.newLine();
        writer.write("Linha 2");

        writer.close();
    }
}
```

``

## Leitura de Arquivos

### FileReader e BufferedReader

- Usados para **leitura de arquivos de texto**
    
- Podem ler caractere por caractere ou linha por linha
    

### Exemplo – BufferedReader

```
import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class FileReaderExample {
    public static void main(String[] args) throws IOException {
        BufferedReader reader = new BufferedReader(new FileReader("output.txt"));

        String line;
        while ((line = reader.readLine()) != null) {
            System.out.println(line);
        }

        reader.close();
    }
}
```

# Fluxos de Bytes

## Conceito

Fluxos de bytes são usados para **arquivos binários**, como:

- Imagens
    
- Áudio
    
- Vídeos
    

Eles trabalham diretamente com **bytes**, não com caracteres.

---

## InputStream

Classe abstrata que representa **fluxo de entrada de bytes**.

### Subclasse: FileInputStream

- Lê bytes de um arquivo
    
- Retorna `-1` ao final da leitura
    

### Exemplo – FileInputStream

```
import java.io.FileInputStream;
import java.io.IOException;

public class FileInputStreamExample {
    public static void main(String[] args) throws IOException {
        FileInputStream input = new FileInputStream("image.jpg");

        int data;
        while ((data = input.read()) != -1) {
            System.out.print(data);
        }

        input.close();
    }
}
```



---

## OutputStream

Classe abstrata que representa **fluxo de saída de bytes**.

### Subclasse: FileOutputStream

- Grava bytes em um arquivo
    

### Exemplo – FileOutputStream

```
import java.io.FileOutputStream;
import java.io.IOException;

public class FileOutputStreamExample {
    public static void main(String[] args) throws IOException {
        FileOutputStream output = new FileOutputStream("data.bin");

        output.write(65);
        output.write(66);

        output.close();
    }
}

```


---

## BufferedInputStream e BufferedOutputStream

### Características

- Usam buffer
    
- Mais rápidos que leitura byte a byte
    
- Ideais para arquivos grandes
    

### Exemplo – BufferedInputStream

```
import java.io.BufferedInputStream;
import java.io.FileInputStream;
import java.io.IOException;

public class BufferedInputExample {
    public static void main(String[] args) throws IOException {
        BufferedInputStream input =
                new BufferedInputStream(new FileInputStream("video.mp4"));

        int data;
        while ((data = input.read()) != -1) {
            System.out.print(data);
        }

        input.close();
    }
}
```

# Diretórios em Java

## Conceito

Diretórios são **pastas** que armazenam arquivos e outras pastas, permitindo melhor organização dos dados.

---

## Criação de Diretórios

### Método `mkdirs()`

- Cria o diretório
    
- Cria diretórios pais ausentes
    

### Exemplo – Criar Diretório

```
import java.io.File;

public class CreateDirectoryExample {
    public static void main(String[] args) {
        File dir = new File("meuDiretorio");

        if (!dir.exists()) {
            dir.mkdirs();
        }
    }
}
```



---

## Listar Arquivos de um Diretório

### Método `list()`

- Retorna um array de `String`
    
- Deve ser verificado contra `null`
    

### Exemplo – Listar Diretório

```
import java.io.File;

public class ListDirectoryExample {
    public static void main(String[] args) {
        File dir = new File("meuDiretorio");

        String[] files = dir.list();

        if (files != null) {
            for (String name : files) {
                System.out.println(name);
            }
        }
    }
}
```



---

## Exclusão de Diretórios

### Regras

- O diretório deve estar vazio
    
- Verificar se existe antes de excluir
    

### Exemplo – Excluir Diretório

```
import java.io.File;

public class DeleteDirectoryExample {
    public static void main(String[] args) {
        File dir = new File("meuDiretorio");

        if (dir.exists()) {
            dir.delete();
        }
    }
}
```

# Java NIO (New Input/Output)

## Conceito

O **Java NIO** oferece:

- Melhor desempenho
    
- Melhor tratamento de erros
    
- Ideal para arquivos grandes
    

---

## Classe Path e Files

### Exemplo – Criar Diretório com NIO

```
import java.nio.file.Files;
import java.nio.file.Path;
import java.nio.file.Paths;
import java.io.IOException;

public class NIOExample {
    public static void main(String[] args) throws IOException {
        Path path = Paths.get("nioDir");

        if (!Files.exists(path)) {
            Files.createDirectories(path);
        }
    }
}
```


---

# Comparação

## java.io

- Mais simples
    
- Tradicional
    
- Usa `IOException`
    

## java.nio

- Mais eficiente
    
- Melhor para grandes volumes
    
- Tratamento de erros mais robusto
