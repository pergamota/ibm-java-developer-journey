# Gerenciamento de Data e Hora em Java

## Conceito

O **pacote `java.time`** fornece uma abordagem **moderna, segura e eficiente** para trabalhar com **datas, horas e fusos horários**, substituindo as classes antigas do `java.util.Date` e `Calendar`.

---

# Classes Principais de Data e Hora

## LocalDate

Representa **apenas uma data** (ano, mês e dia), **sem hora e sem fuso horário**.

### Quando usar

- Data de nascimento
    
- Datas de cadastro
    
- Datas sem relevância de horário
    

### Exemplo – LocalDate

```
import java.time.LocalDate;

public class LocalDateExample {
    public static void main(String[] args) {
        LocalDate today = LocalDate.now();

        System.out.println(today);
    }
}
```

---

## LocalTime

Representa **apenas a hora**, sem data e sem fuso horário.

### Quando usar

- Alarmes
    
- Cronômetros
    
- Horários fixos
    

### Exemplo – LocalTime

```
import java.time.LocalTime;

public class LocalTimeExample {
    public static void main(String[] args) {
        LocalTime now = LocalTime.now();

        System.out.println(now);
    }
}
```


---

## LocalDateTime

Representa **data, hora**, mas **sem fuso horário**.

### Quando usar

- Reuniões locais
    
- Agendamentos internos
    
- Eventos sem fuso
    

### Exemplo – LocalDateTime

```
import java.time.LocalDateTime;

public class LocalDateTimeExample {
    public static void main(String[] args) {
        LocalDateTime meeting = LocalDateTime.now();

        System.out.println(meeting);
    }
}
```

---

## ZonedDateTime

Representa **data, hora e fuso horário**, permitindo identificar um momento exato no mundo.

### Quando usar

- Voos
    
- Sistemas globais
    
- Conversão entre fusos horários
    

### Exemplo – ZonedDateTime

```
import java.time.ZonedDateTime;
import java.time.ZoneId;

public class ZonedDateTimeExample {
    public static void main(String[] args) {
        ZonedDateTime newYorkTime =
                ZonedDateTime.now(ZoneId.of("America/New_York"));

        System.out.println(newYorkTime);
    }
}
```

# Formatação de Datas

## Conceito

**Formatar datas** significa converter um objeto de data em uma **String legível**, seguindo um padrão específico.

---

## DateTimeFormatter

Classe usada para definir **padrões personalizados** de formatação.

### Exemplo – Formatar LocalDate

```
import java.time.LocalDate;
import java.time.format.DateTimeFormatter;

public class DateFormattingExample {
    public static void main(String[] args) {
        LocalDate currentDate = LocalDate.now();

        DateTimeFormatter formatter =
                DateTimeFormatter.ofPattern("dd/MM/yyyy");

        String formattedDate = currentDate.format(formatter);

        System.out.println(formattedDate);
    }
}
```

---

## Exemplo da Vida Real – Data de Nascimento

### Etapas

1. Receber a data como String
    
2. Converter para LocalDate
    
3. Formatar para exibição amigável
    

### Exemplo

```
import java.time.LocalDate;
import java.time.format.DateTimeFormatter;

public class BirthDateExample {
    public static void main(String[] args) {
        String input = "2003-08-15";

        LocalDate birthDate = LocalDate.parse(input);

        DateTimeFormatter formatter =
                DateTimeFormatter.ofPattern("dd/MM/yyyy");

        System.out.println(birthDate.format(formatter));
    }
}
```

# Parsing de Datas

## Conceito

**Parsing** converte uma **String** em um **objeto de data**, permitindo manipulação lógica.

### Exemplo – Parsing com ZonedDateTime

```
import java.time.ZonedDateTime;

public class ParsingExample {
    public static void main(String[] args) {
        ZonedDateTime meeting =
                ZonedDateTime.parse("2024-12-30T15:00:00Z");

        System.out.println(meeting);
    }
}
```

# Fusos Horários (Time Zones)

## ZoneId

Representa um identificador de fuso horário, como `"America/New_York"`.

### Exemplo – ZoneId

```
import java.time.ZoneId;

public class ZoneIdExample {
    public static void main(String[] args) {
        ZoneId zone = ZoneId.of("America/Sao_Paulo");

        System.out.println(zone);
    }
}
```

---

## ZoneOffset

Representa um **deslocamento fixo em relação ao UTC**.

### Quando usar

- Sistemas que exigem precisão
    
- Armazenamento simples de horário
    

### Exemplo – ZoneOffset

```
import java.time.ZoneOffset;

public class ZoneOffsetExample {
    public static void main(String[] args) {
        ZoneOffset offset = ZoneOffset.ofHours(-3);

        System.out.println(offset);
    }
}
```

---

## OffsetDateTime

Representa **data + hora + deslocamento**, sem regras completas de fuso horário.

### Exemplo – OffsetDateTime

```
import java.time.OffsetDateTime;
import java.time.ZoneOffset;

public class OffsetDateTimeExample {
    public static void main(String[] args) {
        OffsetDateTime dateTime =
                OffsetDateTime.now(ZoneOffset.UTC);

        System.out.println(dateTime);
    }
}
```

---

## withZoneSameInstant()

Ajusta a hora para outro fuso, **mantendo o mesmo instante real**.

### Exemplo – Conversão de Fuso

```
import java.time.ZonedDateTime;
import java.time.ZoneId;

public class ZoneConversionExample {
    public static void main(String[] args) {
        ZonedDateTime saoPaulo =
                ZonedDateTime.now(ZoneId.of("America/Sao_Paulo"));

        ZonedDateTime tokyo =
                saoPaulo.withZoneSameInstant(ZoneId.of("Asia/Tokyo"));

        System.out.println(saoPaulo);
        System.out.println(tokyo);
    }
}
```

# ChronoUnit

## Conceito

`ChronoUnit` permite realizar **cálculos temporais**, como diferenças e adições de tempo.

### Exemplo – Diferença entre Datas

```
import java.time.LocalDate;
import java.time.temporal.ChronoUnit;

public class ChronoUnitExample {
    public static void main(String[] args) {
        LocalDate start = LocalDate.of(2024, 1, 1);
        LocalDate end = LocalDate.now();

        long days = ChronoUnit.DAYS.between(start, end);

        System.out.println(days);
    }
}
```
